# browser-enabled-ai-agent
A simple demo of browser use based agent using langchain groq .


### How to run code

- Create conda env

```bash
conda create -p venv python==3.11 -y
```

- Activate the env

```bash
conda activate venv
```

- Install required packages

```bash
pip install -r requirements.txt
```

- Install playwright
```bash
playwright install
```

- command to run bot

```bash
python app.py
```
